### 2.2 常见指令
- `pip install 包名` : 安装包
- `pip install 包名==版本号` :安装对于版本号的包
- `pip uninstall 包名` :卸载对应的包
- `pip show 包名` : 显示包的具体信息，也可用于判断包的安装与否
